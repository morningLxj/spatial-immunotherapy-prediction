# Data Accession Manifest

This directory collects dataset and evidence-source manifests for the PLOS ONE submission branch.

Included files:

- `source_data_manifest.csv`: source-data and dataset manifest.
- `evidence_chain_map.csv`: claim-to-source traceability map.
- `hpa_image_sources.csv`: Human Protein Atlas image-source and license references.

These files support repository traceability and should be interpreted together with the PLOS ONE upload package in `PLOS_ONE_Final_Upload_Slim/`.
