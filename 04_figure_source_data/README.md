# Figure Source Data

This directory contains PLOS ONE figure files and references to the corresponding source-data package.

Included content:

- `PLOS_ONE_Figures_TIF/`: main and supplementary figures prepared as TIFF files for PLOS ONE upload.

Machine-readable figure-source tables are included in `02_processed_source_tables/S1_File_Source_Data_and_Machine_Readable_Tables.zip`.
