# Reproducibility notes

This repository was rebuilt for the conservative PLOS ONE public-data integration version of the manuscript.

The repository intentionally avoids legacy framing that could imply:

- mechanistic proof;
- genetic-instrument-based conclusions;
- individual-level clinical deployment;
- treatment-response biomarker validation;
- in-house experimental validation.

The included scripts verify source-table consistency and summarize processed results from the provided machine-readable tables.
