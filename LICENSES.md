# Licensing and Reuse Notes

## Code and Repository Documentation

Repository code, lightweight integrity checks, and repository documentation are released under the MIT License in `LICENSE`.

## Manuscript and Figures

The manuscript, supplementary material, and figures are provided for scholarly review, citation, and reproducibility inspection. Reuse should follow the final journal license after publication. Until journal publication, do not treat these files as a clinical product, treatment-selection tool, or independently licensed clinical assay.

This repository provides processed source-data tables and traceability documentation for reviewer inspection; it does not change the licensing or access rules of TCGA, GEO, GTEx, Human Protein Atlas, or other source resources.
