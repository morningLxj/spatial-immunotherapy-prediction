# Code

This directory contains lightweight repository checks for the public source-data package.

## Current Script

- `check_source_data.R`: verifies that the machine-readable source-data archive exists, lists its contents, and writes an inventory to `release/source_data_inventory.csv`.

## Scope

The current code is not a complete raw-data-to-figure re-analysis pipeline. It is intended to support repository integrity and reviewer inspection of the processed source-data package.
